# Bosun Papers

This repository stores code, data, and reproducibility artifacts for papers written at Bosun.

Each paper lives in its own top-level directory and includes a paper-specific README with details on the research, files, and usage instructions.

## Contents

- `ai-in-cybersecurity/` - materials for *Artificial Intelligence in Cybersecurity: Reproducible Evidence on Vulnerability Discovery and Security Operations*
- `intervention-penalty/` - materials for *The Intervention Penalty: A Simulation Study of Human Checkpoint Costs in AI Coding Governance*

## How To Use This Repo

1. Open the README inside the paper directory you care about.
2. Follow the instructions in that folder for running analyses, regenerating artifacts, or reviewing supporting data.
3. Treat each paper directory as an independent project unless its README says otherwise.
